# 🎬 YouTube Thumbnail

Uma ferramenta simples em **PHP** para capturar thumbnails de qualquer vídeo do YouTube em diferentes resoluções.

## 🚀 Funcionalidades
- Extrai automaticamente o **ID do vídeo** a partir da URL.
- Exibe todas as versões de thumbnail disponíveis:
  - `default.jpg`
  - `hqdefault.jpg`
  - `mqdefault.jpg`
  - `sddefault.jpg`
  - `maxresdefault.jpg`
- Interface minimalista e fácil de usar.

## 📸 Exemplo de uso
Digite a URL do vídeo no formato:

O sistema exibirá as thumbnails correspondentes:

- Default  
- HQ (alta qualidade)  
- MQ (média qualidade)  
- SD (definição padrão)  
- MaxRes (máxima resolução)
